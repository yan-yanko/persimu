# תרשימי זרימה - תהליכי משתמש

## 1. תהליך התחברות

```mermaid
graph TD
    A[כניסה לדף הבית] --> B{משתמש מחובר?}
    B -->|כן| C[העברה לדף הראשי]
    B -->|לא| D[הצגת טופס התחברות]
    D --> E{אימות תקין?}
    E -->|כן| F[שמירת טוקן]
    F --> C
    E -->|לא| G[הצגת שגיאה]
    G --> D
```
